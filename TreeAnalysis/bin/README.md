Executables
-----------

This directory is supposed to contain the executable for the multi boson analyses.
Right now the package produces the following executables:

- ```eventAnalyzer```

The Analyses currently registered are:

- ```VVXAnalyzer``` --> example, not supposed to be modified.
- ```ZZWAnalyzer``` --> QGC analysis, pp --> ZZW --> 4l2j
- ```ZZSAnalyzer``` --> VV scattering analysis, pp --> ZZjj --> 4ljj
